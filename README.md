Solar System
============
by Geoffrey Mon

A solar system in VPython.
* Includes all eight planets and the sun
* Size is not to scale.
* Distances are to scale.

The without graphics version is for faster calculation of positions.

Textures are by NASA and Björn Jónsson (http://www.mmedia.is/~bjj/).

Created for Coursera Intro to Physics 1 by GeorgiaTech.

Special thanks to Chin Chai Lee and Juan José D'Ambrosio for their help in writing this program.
